## redis-cluster

create redis cluster in docker

### Reference

https://github.com/carwestsam/redis-cluster-in-docker-osx

## Usage

```sh
git clone git@github.com:rhea-so-lab/redis-cluster.git
cd redis-cluster
chmod +x setup.sh
./setup.sh
```
> using port from 20000 ~ 20005

### Warning

it works on macOS
